# Gestore Ricevute in Python




# Descrizione

Questo progetto in Python permette la gestione di ricevute di acquisto e ricevute di ordine.  
Le informazioni vengono salvate su un file di testo e possono essere lette in qualsiasi momento.



# Funzionalità

- Creazione di una ricevuta di acquisto
- Creazione di una ricevuta di ordine
- Salvataggio delle ricevute su file (`ricevute.txt`)
- Lettura e visualizzazione delle ricevute salvate
- Gestione dell’utente tramite classe `Utente`
- Menu interattivo con ciclo




# Struttura del progetto


- `main.py` → contiene il menu e la logica principale
- `utente.py` → classe Utente
- `ricevute.txt` → file dove vengono salvate le ricevute





# Come eseguire il programma


1. Aprire il terminale nella cartella del progetto
2. Eseguire il comando:


```bash
python main.py
